# Robotics-Tools-Resources
A comprehensive List of commonly used Tools and Resources for research &amp; development in wheeled, legged and hybrid robots

## 3-D Modeling Tools
Onshape is a cloud-native, web-based Computer-Aided Design (CAD) platform for creating, sharing, and collaborating on 3D models:
https://www.onshape.com/en/ 

## Simulation Tools:

* ### Coppelia Sim
  https://www.coppeliarobotics.com/
  * An open-source, free-to-use multi-body simulator for rapid prototyping and fast verification of robotics software, digital twins
  * Compatible with remote clients - Python/ C/ C++/ MATLAB/ Rust etc.
